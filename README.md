# Notes Sanctuary

A modern, minimalist notes application built with React, TypeScript, and Vite.

## Features
- **Premium Design**: Serene, distraction-free interface with smooth animations.
- **Three-Column Layout**: Sidebar, Note List, and Editor.
- **Dark/Light Mode**: Toggle between a warm light mode and a deep dark mode.
- **Real-time Search**: instantly filter notes by title or content.
- **Folder Organization**: Categorize notes (System & User folders).
- **Rich Text Support**: (Prototype) Basic rich text editing.

## Getting Started

### Prerequisites
- Node.js (v18 or higher)
- npm or yarn

### Installation

1.  Install dependencies:
    ```bash
    npm install
    # or
    yarn
    ```

2.  Start the development server:
    ```bash
    npm run dev
    # or
    yarn dev
    ```

3.  Open your browser at the URL shown (usually `http://localhost:5173`).
